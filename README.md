### 🚀 What I'm Working On

- **Asset Management System v2.0** — full asset-lifecycle platform: a NestJS + GraphQL API on PostgreSQL/Prisma, a Next.js 14 frontend, Redis caching, Socket.IO real-time updates, and Gemini-powered insights.
- **Boardmeets / e-Board** — multi-tenant board-governance SaaS (Django · React · PostgreSQL · Celery · WebSockets) with layered RBAC, eCitizen OAuth2, OCR search, and digital signatures.
- **QAForge** — a QA automation platform for test orchestration and reporting across products.
